# Pokemon App Base on Framework 7 Bundle
## a rewrite of [ The original Pokemon App](https://github.com/survos-sites/pokemon) using [FW Bundle](https://github.com/survos/fw-bundle)

## Installation

```bash
git clone git@github.com:survos-sites/fwpokemon fwpokemon && cd fwpokemon
composer install
symfony server:start -d
symfony open:local
```
